# Titanic Survival Prediction 🚢

An end-to-end Data Science project exploring the famous Titanic dataset from Kaggle.

The goal of this repository is not only to build a predictive model, but also to demonstrate the complete Data Science workflow—from understanding the data through Exploratory Data Analysis (EDA), handling missing values, engineering meaningful features, and finally training and evaluating machine learning models.

> **Project Status:** 🚧 In Progress

---

## Project Objectives

- Perform an in-depth Exploratory Data Analysis (EDA)
- Understand the relationships between passenger characteristics and survival
- Handle missing values appropriately
- Clean and preprocess the dataset
- Engineer new features to improve model performance
- Train and compare multiple Machine Learning models
- Evaluate model performance using appropriate metrics
- Document every step for educational purposes

---

## Dataset

This project uses the **Titanic - Machine Learning from Disaster** dataset provided by Kaggle.

The dataset contains information about passengers aboard the RMS Titanic, including:

- Passenger demographics
- Ticket class
- Fare paid
- Cabin information
- Family relationships
- Embarkation port
- Survival outcome
